SourceInfo Class¶

class SourceInfo¶

Source metadata information.

Fields are the empty string or zero if the information is not available.

Variables:
  • title – Title
  • author – Author
  • copyright – Copyright statement
  • comment – Comment
  • album – Album name
  • year – Year
  • track – Track number
  • genre – Genre

Since pyglet 1.2
